# Mini Project Big Data

## Overview
This repository contains a Big Data mini project developed using Python and Google Colab.
This project performs data processing, analysis, and visualization to obtain insights from the provided dataset.

## Project Description
The project includes several stages:

- Data loading
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Data analysis
- Model evaluation

## Dataset
The dataset used in this project is stored in the `dataset` folder.
The dataset is used as the main source for the analysis process.

## Tools & Technologies
- Python
- Google Colab
- PySpark
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
